# CodeAlpha Data Analytics Internship - Task 1

## Project Name: Web Scraping Data

### Overview
For this task, I built a custom Python web scraper to extract product data from an e-commerce website. This project fulfills the requirements for Task 1 of the CodeAlpha Data Analytics track.

### Repository Structure
- `Web_Scrapping.py`: Core Python script using BeautifulSoup to handle HTML navigation.
- `scraped_books_dataset (2).csv`: Clean, custom dataset featuring scraped book names, pricing, and availability metrics.
